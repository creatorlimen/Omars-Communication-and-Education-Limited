## Quick Start

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up your environment variables
4. Run the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `/paths` - Application templates and prompt configurations
- `/src/app` - Next.js application routes and API endpoints
- `/src/components` - Reusable React components
- `/src/lib` - Utility functions and context providers
- `/src/interfaces` - TypeScript type definitions

## Creating New Projects

1. Navigate to `/paths`
2. Choose a template from the available paths
3. Use the Composer feature to scaffold your project
4. Customize the prompt templates as needed

## Technologies

- **Frontend**: React, Next.js 14, TailwindCSS
- **Backend**: Next.js API Routes
- **Authentication**: Firebase Auth
- **Database**: Firebase Storage and Database
- **AI Services**: 
  - OpenAI - For chat and text processing
  - Anthropic - For advanced language tasks
  - Replicate - For image generation
  - Vercel AI SDK - For streamlined AI integration
